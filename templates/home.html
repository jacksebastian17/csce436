CSCE 436 Group 2 - Grocery Helper (object detection)

About the app

Grocery Helper is an app that uses computer vision to help visually impaired people identify objects while shopping by providing audio output. Simply point your phone's camera at an object, and the app will tell you what it is.

Grocery Helper app uses computer vision to identify objects in real-time using the OpenCV library and the YOLOv4 object detection algorithm. OpenCV is a powerful computer vision library that provides tools for image processing, feature detection, and object tracking. YOLOv4 is a state-of-the-art object detection algorithm that uses a deep neural network to detect objects in images and videos. The algorithm has been trained on a large dataset of images and is capable of detecting a wide range of objects, including fruits, vegetables, and other grocery items. The YOLOv4 algorithm is implemented using the Darknet framework, which is an open-source neural network framework written in C and CUDA. The Grocery Helper app uses the pre-trained YOLOv4 weights to detect objects in real-time, and provides an audio output to assist visually impaired users in identifying objects while shopping.

How to use the app:

  1. Open Grocery Helper app on your phone.
  2. Press the green "Start" button.
  3. Make sure you allow camera privileges on the device you're using it on.
  4. Point your phone's camera at an object you want to identify.
  5. Wait for the app to provide an audio description of the object.
  6. Repeat steps 4-5 for any other objects you want to identify.
  7. When you're finished shopping, click the "Back" button to end the session and go back to the home page.
